## 2) .ini ファイルを編集する
このファイルは 3) で起動するファイルと同階層にある必要あり
- url - スレURL
- interval_seconds - 書き込み間隔 できるだけ大きくして書き込み頻度下げる方がいい
- from - 名前 空でもいいけどNGしやすいようになんか決めといたほうがいいかも
- mail - メール欄 sageてはいけないらしいので注意
- message - 書き込み内容 公序良俗

## 3-a) .py ファイルを実行する
pythonのセットアップができていることが前提<br>
pythonなんか入れてねーよって場合は 3-b) へ

```pip install selenium```
でツール実行に必要なものをインストール（裏でブラウザ動かす用のやつ）

.py をダブルクリックで起動 コマンドプロンプトみたいな画面が出てくるので最小化して放置でOK

終わりたいときはコマンドプロンプトみたいな画面で Ctrl+C 連打してください

## 3-b) .exe ファイルを実行する
python環境なくてもいい版<br>
pyinstallerで変換しただけなので _internal フォルダが同階層にある必要あり

.exe をダブルクリックで起動 コマンドプロンプトみたいな画面が出てくるので最小化して放置でOK

終わりたいときはコマンドプロンプトみたいな画面で Ctrl+C 連打してください
